# blusowon.github.io
Progetto HTML e CSS Start2impact
Project from the html and css course on Start2impact University.

Ho voluto mantenere il più semplice possibile questo progetto essendo il primo che faccio in piattaforma. 
Avendo cominciato il percorso col master Front-End Development e decidendo poi di passare a UX/UI Design, ho scelto di finire prima la guida ed 
il progetto HTML e CSS, perciò ci sono un portfolio molto scarso e pochi riferimenti social. Mi sono concentrata di più sul finire il progetto, 
avendo iniziato e cestinato molti altri progetti prima di questo, rischiando un blocco totale nel proseguimento del master. Nonostante conosca 
e abbia studiato Bootstrap, recependo le regole principali, ho volutamente scelto di non metterlo per il suddetto motivo, insieme ad una media 
query molto semplice e concentrata sul rendere "comoda" l'esperienza; difatti, ho strutturato le varie pagine verticalmente. Sempre per il motivo 
sopra, anche l'accessibilità è stata tralasciata, concentrandomi più sulla resa grafica e creando quindi contrasti tra testo e background o immagini, 
cambiando colore sugli elementi all'hover dei link, ecc. Per il resto, il progetto risulta molto semplice, in quanto desiderosa di continuare 
il master e arricchire piano piano questo portfolio sia a livello di contenuti personali, sia di codice, magari implementando bootstrap e sass.
